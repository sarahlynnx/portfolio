# Sarah Olson — Portfolio

Personal portfolio website, full-stack developer and founder of [LynnX](https://lynnx.dev). Built with Next.js 16, TypeScript, Tailwind CSS v4, and Motion.

## Stack

- **Framework** — Next.js 16 (App Router)
- **Language** — TypeScript
- **Styling** — Tailwind CSS v4
- **Animations** — Motion (Framer Motion)
- **Deployment** — Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Sections

- **Hero** — Animated name reveal with role, location, and CTAs
- **About** — Professional summary and background
- **Skills** — Categorized tech stack
- **Experience** — Work history with timeline layout
- **Projects** — Featured projects with links
- **Certifications** — freeCodeCamp, MongoDB University, Educative.io
- **Contact** — Contact form and social links
